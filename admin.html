⚙️ Google Sheets Integration

📋 Setup Instructions

  1. Open Google Sheets → New spreadsheet
  2. Name sheets: Orders, Contacts, BakeSessions
  3. Go to Extensions → Apps Script
  4. Paste the Google Apps Script code from google-apps-script.js
  5. Click Deploy → New deployment → Web App
  6. Set access to "Anyone" and copy the URL
  7. Paste the URL in the field above and click Save
